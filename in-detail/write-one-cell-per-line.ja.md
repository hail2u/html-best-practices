# 1行に1つのセルを書く

長い行は把握しづらいです。

悪い例:

    <tr>
      <td>General</td><td>The root Element</td><td>Sections</td>
    </tr>

良い例:

    <tr>
      <td>General</td>
      <td>The root Element</td>
      <td>Sections</td>
    </tr>

「[リスト項目は1行に1つずつ書く][1]」を参照してください。
